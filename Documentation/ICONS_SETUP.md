### Logo Guidelines:
- Use a simple, recognizable icon
- Ensure good contrast (icon vs background)
- Avoid text-heavy logos (hard to read small)
- Use a circular or rounded square design
- Maintain consistent branding

## ğŸ”§ Troubleshooting

**Icons not showing after install?**
1. Clear browser cache
2. Uninstall PWA
3. Rebuild: `npm run build`
4. Reinstall PWA

**Icons appear blurry?**
- Make sure you're using the exact dimensions
- Use PNG format (not JPG)
- Ensure high-quality source image

**Icons don't appear in manifest?**
- Check file names match exactly
- Ensure files are in `public` folder
- Rebuild the app
